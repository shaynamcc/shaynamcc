## Featured Projects

### Fully Reproducible Sales Data Analysis
- This project provides summaries and visualizations for January sales data.
  Using customer lookup information and transaction records, the analysis 
  produces a CSV file with region + category breakdowns for sales, and a bar
  chart showing total revenue by region. This repository is organized with
  separater folders for data, scripts, and results to make it easy to 
  navigate and reproduce.

### NYC 311 Service Request Analysis on AWS
- This project analyzes around 200,000 NYC 311 service requests and builds a 
  predictive model to forecast complaint resolution times across agencies and boroughs. 
  Using AWS cloud infrastructure (S3, SageMaker) and machine learning (Linear Learner), 
  the project processes large-scale complaint data, performs exploratory analysis, and 
  develops a regression model. The work demonstrates end-to-end cloud data engineering 
  with stakeholder-focused predictive insights.

### Sakila DB Revenue Analysis by Country
- This project analyzes rental revenue by country using the Sakila rental database, 
  answering the key question: "Which countries generate the highest rental revenue 
  and how should resources be allocated?" The analysis demonstrates advanced SQL 
  techniques including joins, aggregate queries, CTEs, and window functions to 
  provide data-driven insights for resource allocation in a rental business context.
